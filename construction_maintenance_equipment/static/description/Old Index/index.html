Construction and Contracting with Equipment & Maintenance

This module will add Odoo standard Maintenance and Equipment menus under Job Application and link Maintenance with Job order and Projects.

Main Features:

  • Link Projects and Job Order with Maintenance Request.
  • For more details please see https://apps.odoo.com/apps/modules/11.0/asset_mro_maintenance_management/ which is dependant app.
  • Bring all the odoo standard menus of Equipment & Maintenance under jobs.
  • For more details please watch below screenshots and Video.

Menus Available:

  • Jobs
    • Maintenance
      • Maintenance Requests
      • Maintenance Equipments
    • Configuration
      • Maintenance
        • Maintenance Checklists
