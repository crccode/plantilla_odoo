Project Job Orders Request from Website

This module allows your customer/visitor to create a job order or work order from your website.


Main Features:
  • Anyone (Customers / Visitors) can create a job order or work order from your website.
  • Allow your customers to have a portal in My Account to view a list of job orders and print job orders.
  • Job orders / Work orders linked with Project so if customers fill Project code [optional] while submitting a request from the website then the job order will be linked to specific Project.
  • Instance email notification to customer by email when job order will be created.
  • Print Job Order PDF from backend as well by your internal team.
  • Allow you to configure the job category in the backend and allow your customer to select from the request form.
  • Allow your team to send by email to your customer from backend and attach a report of job order in backend.
  • Integrated with Material Requisitions by Employees/Users so you can check material_purchase_requisitions app for more details about its features. Please click here :https://www.odoo.com/apps/modules/13.0/material_purchase_requisitions/.
  • For more details please watch the video and check below screenshots.
