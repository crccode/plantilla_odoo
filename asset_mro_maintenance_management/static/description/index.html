Asset MRO - Equipment Maintenance Management

MRO - Equipment and Maintenance Request Management - Odoo Module

This app allows you to create "Material Requisition" and "Job Order/Work Order" from the maintenance request form. It also allows you to configure Planning of materials and Checklists on Equipment which are going to be used while creating "Maintenance Request" of Equipment.

Main Features:

  • Allow Equipment/Maintenance Manager to create a checklist(s) in Equipment configuration and checklists will be used during Maintenance requests and process.
  • Allow you to configure Materials/Items which are going to be used in Maintenance Request on Equipment form.
  • Allow you to configure the checklist(s) which will be used in Maintenance requests on the Equipment form.
  • Auto populate checklist(s) and Planned Material(s) on Maintenance request.
  • Allow you to create a Job Order from Maintenance Request. Please see the https://apps.odoo.com/apps/modules/13.0/website_job_workorder_request app for more details.
  • Allow you to create material requisition requests. For more details about https://apps.odoo.com/apps/modules/13.0/material_purchase_requisitions app for more details.
  • Adapt odoo Maintenance PDF report to show Checklist(s) and planned material(s).
